Patients with lesions to particular brain regions lose specific capacities not because the magic substance was damaged, but because the functional organization was disrupted. Moreover, functionalism makes clear, testable predictions: if we can replicate the relevant functional organization of a brain in another medium, that system should be conscious. I welcome them, but I want to note upfront that many of these thought experiments smuggle in dualist intuitions disguised as logical arguments. We should not mistake the stubbornness of an intuition for evidence of a metaphysical gap.
